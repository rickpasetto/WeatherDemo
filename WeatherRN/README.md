# US Weather - React Native

A simple weather app showing current conditions for major US cities, built with React Native and Expo.

## Features

- Shows weather for 10 major US cities
- Add new cities via search
- Uses free Weather.gov API (no API key needed)
- Pull-to-refresh

## Getting Started

```bash
# Install dependencies
npm install

# Start the app
npx expo start
```

Then scan the QR code with Expo Go (Android) or Camera app (iOS).

## Tech Stack

- React Native with Expo
- Weather.gov API for weather data
- Nominatim (OpenStreetMap) for geocoding
